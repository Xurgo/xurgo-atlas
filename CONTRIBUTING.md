# Contributing

Thanks for checking out Xurgo Atlas.

Atlas is an early public project. Helpful feedback includes bug reports, setup feedback, MCP client compatibility notes, docs fixes, clear reproduction cases, and small targeted tests.

## Issues

Please open an issue for bugs, setup problems, confusing docs, or MCP compatibility notes.

## Pull requests

Small docs fixes and targeted test improvements are welcome.

Please open an issue before starting larger or behavior-changing PRs, especially changes involving root/worktree safety, storage, MCP tool contracts, managed-doc write/export behavior, release files, or filesystem security.

## Release and automation

Validation CI is welcome. Release, version, and npm publish changes should start with an issue and follow maintainer guidance.

Please do not add publish-on-main automation, PR-triggered publishing, npm publish tokens, or release secrets without explicit maintainer direction. Local git hooks are optional for maintainers and are not required for public contributions.

## Security

Please do not report security issues in public issues. See [SECURITY.md](SECURITY.md).
